## CAO
> making charity more transparent

### Run
```
$ cd client
$ npm install; npm run start
```

### Truffle & Ganache
Make sure to run `ganache-cli -p 7545`, then do:
- `truffle compile`
- `truffle migrate`
- `cd client & npm test`
- `npm run start`
